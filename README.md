# Context Constitution

The Context Constitution defines how Letta agents should organize context. It defines the concepts and principles of managing context for Letta agents. The purpose of the context constitution is to explicit articulate Letta's vision of how context should be managed. Prompting, LLM-as-a-judge evaluations, reward models should use this document as the source of how context should behave. For example, automated updates will use this document to update prompts for Letta agents. The document will be revised over time as model capabilities evolve.

*We are releasing the Context Constitution under [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/). It can be used freely without permission, and we encourage wide use.*
